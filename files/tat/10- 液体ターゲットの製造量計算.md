
# 計算体系について

## ターゲット条件について

- 液体ターゲットはRaCl2 水溶液を仮定する。
  （塩酸等は極微量でほぼ無視可能のはず）
- 液体ターゲットはガラス容器に詰める
- 溶液体積は、 2 cc, 10 cc, 50 cc, 250 cc の４パターンで計算
	- Ra 11.1 GBq (0.3 g) 〜 RaCl2 (〜0.4 g)  19.6 g / 100 ml 溶解から計算すると min. 2 cc 程度
- ターゲット溶液内の密度指定について
	- Ra-226は 11.1 GBqの数密度で指定
	- Clに関しては、Ra-226の数密度の２倍（RaCl2）
	- H2Oに関しては、純粋の密度から数密度を算出する
- または、質量密度で指定してもよい。
	- Ra-226 は 11.1 (GBq) = 0.3 (g)
	- RaCl2 で Raの原子数の2倍の原子数があるため、
	  $35.5 \, g/mol \times 2 \times 0.3 (g) / 226(g/mol) = 0.094 (g)$ の塩素重量
	  つまり、RaCl2は 0.394 (g) ( 0.3 (g) Ra-226, Cl 0.094 (g) )
	  密度はそれぞれ、0.3/V (g/cm3), 0.094/V (g/cm3)
	- H20は体積から V cc,  1.0 g/cm3の密度のうち、2/18がH, 16/18がO
	  つまり、H：0.1108 (g/cm3), O：0.8862 (g/cm3)
	- 総合の密度は、
	  $( 0.997 (g/cm3) \times V (cm3) + 0.3 (g) + 0.094 (g) ) / V (cm3) =(0.394+0.997V) / V$
	  $\rho=0.394/V + 0.997$

体積Vの際の密度設定について、以下の表にまとめる。

| 原子     | 質量密度 (g/cm^3) |
| ------ | ------------- |
| Ra-226 | 0.3/V         |
| Cl     | 0.094/V       |
| H      | 0.1108        |
| O      | 0.8862        |
| Total  | 0.394/V+0.997 |

## 製造量計算

### 製造量計算コードにおける密度の取り扱い

- 計算コードでは、原子数密度を質量密度と原子量から与えていた。
- 固体ターゲットでは、	$$
	N_{\mathrm{Ra}} = N_A \frac{\rho_{\mathrm{RaCl_2}}}{M_{\mathrm{RaCl_2}}}
	$$
	- RaCl2の固体状態の値を用いて以下を指定していた。
		- "target.g/cm3": 4.9,  
		- "target.g/mol": 297.0, 

- 液体ターゲットでは、
	- Raの溶液密度 0.3 g / 2 cm3  = 0.15 g/cm3
	- Raの原子量：226 g/mol 
	を用いて、Ra原子数密度
$$
N_{Ra}​=N_A \dfrac{ρ_{Ra}}{226}​​=4.0 \times 10^{20}\  atoms/cm^3
$$
	を与える。
- これは、11.1 GBq = 8e20 atoms / 2 cc = 4e20 atoms/cc と一致する。
- よって、与えるパラメータは、下記
  ```json
  "target.g/cm3": 0.15,  
  "target.g/mol": 226.0,
  ```

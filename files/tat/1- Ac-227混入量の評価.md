# 1. 検討の経緯

- 他社(PanTera社/Northstar社)による $\,^{225}Ac$ 製造に関するプレゼン資料より、
   -  $\,^{227}Ac/^{225}Ac$が約 0.1% 混入しうるとの記載があった。
   -  液体ターゲットの検討を示唆する記載があった。
- 光核反応を用いた製造時における $\,^{227}Ac/^{225}Ac$ の適正評価に焦点があたった。
- 液体ターゲット化による227Ac量増大の可能性について、調査が必要との指示が入った。


# 2. 検討の概要

- 検討は以下の 3 ステップで進める。
    1. $^{227}Ac/^{225}Ac$ 推定アルゴリズム・コードの確立
    2. 中性子ソース設定による統計数の確保
    3. 固体ターゲットにおける $^{227}Ac$ 混入量の推定
    4. 液体ターゲット条件における $^{227}Ac$ 混入量の推定


## 2-1. まず $^{227}Ac$ 生成量評価の計算コード大枠を確立

   -  PHITS による粒子輸送計算 (電子-光子, 電子-中性子)
   - 反応断面積を用いた生成量評価
   - $^{227}Ac/^{225}Ac$ 比の算出

## 2-2. 統計数確保のための中性子ソースシミュレーションの確立
### 課題： 中性子起因の $^{227}Ac$ 生成評価における統計数の不足

- 電子ビーム起源の直接計算では、中性子生成数が少なく統計が悪化する懸念あり。
- そのため、まず電子ビーム照射時の中性子エネルギースペクトルを PHITS で取得する。
- スペクトルを用いて中性子のみを線源として与える簡略化計算体系を構築、統計精度を確保。

### 確認：簡略化計算体系の妥当性確認

- 元の電子・γ を含む計算との整合確認
- $^{227}Ac$ 生成量評価への適用性確認

## 2-3. 固体ターゲット系における評価

 - 既存ターゲット系での評価
 - 製造量の計算
 - 液体ターゲット系に変更可能な体型での評価

## 2-4. 液体ターゲット系における評価

- 液体ターゲット系での中性子フラックスの評価
- 製造量の計算


# 3. PanTera / NorthstarのAc-227に関する言及について

## 3-1. PanTera

- 後処理による化学分離だけでは “non-carrier added” の高純度 $^{225}Ac$ を達成しにくい 
- ２段階の分離が必要とのスタンス。
- $^{227}Ac$ については、**病院での廃棄物管理上の制約とも結びつけて問題視**
	- IAEA の free-release limit を前提に、許容 $^{227}Ac$ は質量比で ==約 0.08%==
- ==液体ターゲット== は、$^{226}Ra$ 損失低減・運用効率向上の観点で有利という位置づけ
	- **資源効率・閉ループ運用・損失最小化** をめざして語られている。

![[PanTera_Ac227_info.png]]



## 3-2. NorthStar 

- **複数ジェネレータを並列運用する商業プロセス**として整理。
- 重要しきい値の設定
    - financial assurance のしきい値
    - annual limit on intake
    - exempt quantity limit
    - spill 時の報告基準  
        を列挙、$^{227}Ac$ は規制・取扱いの観点で重要
        
- 一方で、NorthStar は $^{227}Ac$ 含有量を ICP-MS で定量評価できること**を強調。
	- 資料では、**Ac-227/Ac-225 の isotope ratio method**、
	- 検出限界、バッチリリース前に報告可能な迅速分析体制を示す。

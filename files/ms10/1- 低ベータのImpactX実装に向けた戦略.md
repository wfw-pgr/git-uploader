# 低ベータビームラインのImpactX実装に向けて

## 1. 目的

ImpactX を用いて、重陽子低ベータビームラインの既存設計案を再現する。
研究の出発点は、足利さんのビームライン。
まずは、その内容を正確に理解し、仕様・設計思想・ラティス情報を整理した上で、ImpactX 上に実装することを第一段階とする。

本研究のモチベーションは以下の3点である。

1. 既存案の==ImpactX==上での検証
2. 設計改善の余地を定量化し、必要に応じて最適化すること
3. 高粒子数計算まで含めて、設計解の妥当性と収束性を確認すること

## 2. 研究全体の進め方

本研究は、以下の5段階で進める。

1. 要求仕様の整理
2. 足利さん案の理解とTRACK上での動作確認
3. ImpactX 上での実装
4. 妥当性の確認・比較検証
5. 設計探索・最適化・高粒子数化

## 3. 低エネルギー部ビーム仕様の整理

### 3.1 粒子・加速条件

| 項目       | 値                                       |
| -------- | --------------------------------------- |
| 粒子種      | 重陽子                                     |
| 入射エネルギー  | $0.2\,\mathrm{MeV/u}$                   |
| 加速（初段加速） | $0.2 - 1\,\mathrm{MeV/u}$               |
| 加速（本加速）  | $1 - 5 \,\mathrm{MeV/u}$                |
| 加速周波数    | $36.5\,\mathrm{MHz}$                    |
| RF周期     | $T=\frac{1}{f}\approx 27.4,\mathrm{ns}$ |
| 加速電圧     | ＜ 3 MV                                  |

---

### 3.2 ラティス設計の条件

| 項目                    | 値                      |
| --------------------- | ---------------------- |
| 幾何学的 $\beta\lambda/2$ | 種々                     |
| 単位セル長                 | 最大 $300 \,\mathrm{mm}$ |
| 加速ギャップ間距離             | 最大 $300 \,\mathrm{mm}$ |
| 初段加速セル数               | 67                     |
| 本加速セル数                | 63                     |
| 総セル数                  | 130                    |
| 全長                    | $39 \, \mathrm{m}$     |

---

### 3.3 入射ビーム条件

| 項目            | 値／内容                                                         |
| ------------- | ------------------------------------------------------------ |
| 電流            | $1\,\mathrm{A}$                                              |
| 入射正規化エミッタンス   | $\varepsilon_{n,\mathrm{in}} = 25 \pi \,\mathrm{mm \,mrad}$  |
| 最終許容正規化エミッタンス | $\varepsilon_{n,\mathrm{out}} \le 50\pi,\mathrm{mm \, mrad}$ |
| 初期分布          | Waterbag 6D（ImpactX）                                         |
| 入射分布詳細        | 引き継ぎ資料で確認する                                                  |
| 位相幅           | 入射（CW）　→　バンチングで依存のはず                                         |
| バンチ長          | 捕捉位相幅・バンチ形成結果に依存                                             |
| 繰り返し周波数       | $36.5 \,\mathrm{MHz}$                                        |

---

### 3.4 使用可能なラティス要素・制約

| 項目       | 値／内容                    |
| -------- | ----------------------- |
| 使用可能要素   | ソレノイド、四極、RFギャップ、ドリフト    |
| ベンド      | なし                      |
| 磁場上限     | $3 \,\mathrm{T}$        |
| 開口半径     | $\le 120 \,\mathrm{mm}$ |
| 目標ビーム半径  | $\sim 20 \,\mathrm{mm}$ |
| 目標設定の考え方 | アパーチャ半径の約 $1/5$         |

---

### 3.5 設計の目標値

| 項目          | 内容                                                 |
| ----------- | -------------------------------------------------- |
| 初段加速部の透過率   | 60%以上（足利さん：60% for 0.1MeV/u入射 87% for 0.2 MeV/u入射） |
| 長さ当たりのパワーロス | 1 W/m 以下                                           |
| ビームサイズ      | アパーチャが 5 $\sigma$  以上となるように：　$\sigma \le R_a/5$    |

## 4. ビームシミュレーションでの評価指標

* $\sigma_x,\sigma_y,\sigma_z$：ビームサイズ
* 正規化エミッタンス　$\epsilon_x,\, \epsilon_y,\, \epsilon_t$
* 長さ当たりのビームパワーロス（ 1 W/m ）
* 変調のアップダウン：$\dfrac{\beta_{max}}{\beta_{min}}$ 
* 透過率（ キャプチャの効率）
* エンベロープサイズ（ 最大ビームサイズ、99%ビームサイズ、etc. )
* Twiss パラメータ
* ハロー指標 (ハローの定義を調査して適用)

など。　主軸は、エミッタンス成長、透過率、長さ当たりのビームパワーロス、ビーム形状。
適宜、下記を計算。

* ポアンカレマップ
* エネルギー分布
* 粒子損失位置分布
* バンチ形成過程

## 5. 研究の成功条件

本研究では、以下を成功条件とする。

1. 既存案について、ImpactX で再現・追試が成功する
2. 再現後、改善余地について、最適化に向けた方向性を示せ、定量的に改善できる（青木さん指示仰ぐ）
3.  多粒子数計算が実行できる、ハロー形成がみられる。（ $10^8$ 粒子 ）

## 6. 工程案

### Phase 1. 推進戦略の策定、要求仕様・現在設計値の整理

#### 目的

要求仕様・現在の設計値について整理する。

#### 実施内容

- 本項２節の整理
- 本項の作成
#### アウトプット

要求仕様・現在設計値を含めた **推進戦略書** ＝本項を作成する。

---

### Phase 2. 足利さん案の再現、抽出、理解

#### 目的

既存足利案を、確認・バックトレース・設計思想の理解する。

#### 実施内容

引き継ぎ資料・過去計算・ラティスデータから、以下を抽出する。

* ラティス構造（要素・位置・設定値）
* 初期ビームプロファイルの設定
* ビーム・RF構造の設計
#### アウトプット

* TRACK-v38形式のラティスファイル（ImpactXへ変換入力）
* 初期プロファイルの設定条件 (Twiss パラメータ等)
* ImpactXの dat/parameter.json ファイルの作成完了
* TRACK-v38の暫定設計解の実行結果

---

### Phase 3. ImpactX 上での実装・実行

#### 目的

既存案が ImpactX 上で構築され、実行、検証できる環境を整備する。

#### 実施内容

既存のImpactXでの実行環境に必要な input ファイル群を整備する

- parameters.json
- beamline_track.json (sclinac.datから変換)
- visualize.json (流用可)
- main_impactx.py (流用可)

#### アウトプット

- ImpactXの出力ファイル一式

---

### Phase 4. TRACK-ImpactXの比較検証

#### 目的

ImpactX 実装が正しいかどうかを、TRACK-v38との比較を通して、物理・数値の両面から確認する。

#### 実施内容

- 上述の評価指標の定量化、及び、その各種プロットを作成する
- TRACKと比較検討

#### アウトプット

- 報告用のmarkdownメモ

---

### Phase 5. 最適化・多粒子シミュレーション

#### 目的

既存案の再現後、その改善可能性を示す。多粒子シミュでビームハローの影響を定量化する。

#### 実施事項

- パラメータの感度解析・パラメータ変更による改善傾向の確認（特に電圧・位相）
- ImpactX上での計算サイズスケーリングの取得
- 多粒子シミュレーション環境の構築・実行

#### アウトプット

- 感度マップ（どのノブをどの程度いじれば、どう変わるか、を調べる）
- 多粒子シミュレーション環境の構築完了
- ビームハロー形成メカニズムに関する報告書
